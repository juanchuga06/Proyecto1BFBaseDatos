# Proyecto del Primer Bimestre de Fundamentos de Bases de Datos

- EPN - 2025B
- GR1SW
- Integrantes: Bryan Ayala, Juan Chugá

## Tema: “Sistema De Gestión De Citas Médicas En Consulta Externa Para el IESS”

Este repositorio contiene todos los elementos necesarios para la implementación de una base de datos que soporta el ingreso y salida de datos de un sistema de gestión de citas creado para ser usado en un centro médico del IESS. La aplicación fue desarrollada en Java y la base de datos se manejó en MySQL 8.0.

## Estructura del proyecto

La estructura de este proyecto es la siguiente. A continuación se detallan los ficheros más importantes:

- **Database:** aquí se encuentran los scripts SQL que incluyen el DDL de la base de datos y una muestra de inserciones en las tablas, por si se desea comenzar a utilizar la base de datos con datos de ejemplo.  
- **Design:** contiene las imágenes en formato `.png` de los modelos entidad–relación, relacional y físico.  
- **Documents:** incluye los archivos `.pdf` del manual y la presentación creada para el proyecto.  
- **.Idea:** librerias propias de un proyecto en Java en IntelliJIDEA.  
- **Lib:** librerías y dependencias propias de un proyecto en Java.  
- **Src:** código fuente en Java implementado para la aplicación.
